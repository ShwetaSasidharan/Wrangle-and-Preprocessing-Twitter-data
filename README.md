# Wrangle-and-Preprocessing-Twitter-data

Designing efficient regular expressions in order to extract the data from the dataset.
Wrangle the text from Json data files to XML data files. Included only English tweets used Langid library to filter tweets from other languages. Included unique tweet id's. Parsed emoji data using encoding and decoding techniques. 

Analyzing textual data, i.e., converting the extracted data into a proper format. Program to preprocess a set of tweets and convert them into numerical representations (which are suitable for input into recommender-systems/ information-retrieval algorithms). Includes N-grams, Bi-grams, unigrams. Created sparse matrix using countvectorizer.
